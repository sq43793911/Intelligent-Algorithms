# Intelligent-Algorithms
 
 
1. TSP：使用了4 种常⻅的最优化智能算法(遗传、蚁群、免疫、禁忌表算法) 对旅行商问题(TSP)进行分析，比较了各类算法的优缺点以及应用价值

        ACO-TSP：蚁群算法求解TSP
        
        GA-TSP：遗传算法求解TSP
        
        Immu-TSP：免疫算法求解TSP
        
        Tabu-TSP：禁忌表算法求解TSP
    
2. 成本计算模型：使用遗传算法对工厂订货仓储成本管理进行了最优化分析。

3. KNN-and-Cluster：基于Matlab的分类学习工具箱，研究并实践了数据挖掘中常用的kNN(k最邻近分类算法)和聚类分析的算法。
